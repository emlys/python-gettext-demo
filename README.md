# python-gettext-demo
Minimal example of python gettext in an application, including packaging and language switching at runtime

Project structure:
```
a/
|-- __init__.py
|-- b.py
|-- c.py
|-- interface.py
|-- locales/
|   |-- ll/
|       |-- LC_MESSAGES
|           |-- messages.mo
|           |-- messages.po
|-- setup.py
|-- pyproject.toml
```
